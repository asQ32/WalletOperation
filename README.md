# Wallet Service

Простой сервис для управления кошельками с поддержкой операций пополнения и снятия средств.

---

## Описание

Проект реализует:
- REST API для получения операций кошелька.
- Сервис для обработки операций (`deposit`, `withdraw`).
- Проверку баланса с выбросом исключения `InsufficientFundsException` при попытке снять больше средств, чем есть на счете.
- Unit-тесты для проверки логики сервиса.

---

## Технологии

- Java 17
- Spring Boot 3.2
- Spring Data JPA
- PostgreSQL (или любая другая поддерживаемая база данных)
- JUnit 5
- Mockito
- Maven

---

## Запуск проекта

1. Клонировать репозиторий:
   
```bash
git clone <URL_вашего_репозитория>
```
   
2. Собрать проект:
mvn clean install

3. Запустить приложение:
mvn spring-boot:run
